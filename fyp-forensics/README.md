# ForensicX: Unified Analysis Framework

**ForensicX** is a digital forensic framework designed to unify **memory**, **log**, and **detection-based analysis** into a single automated workflow.  
The project aims to improve the efficiency and accuracy of digital investigations by integrating multiple forensic data sources within one platform.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Project Overview

ForensicX automates evidence extraction, correlation, and reporting from multiple forensic data sources such as:
- **Volatile Memory (RAM)**
- **System Logs (Sysmon EVTX)**
- **Detection Rules (YARA)**

The system reduces manual correlation tasks, helping investigators quickly identify **Indicators of Compromise (IOCs)**, reconstruct **attack timelines**, and generate **standardized forensic reports**.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Objectives

1. Develop a unified framework that integrates memory, log, and detection data.
2. Automate event correlation and IOC extraction using process and timestamp matching.
3. Generate forensic reports (TXT, JSON, PDF) and provide optional dashboard visualization.
